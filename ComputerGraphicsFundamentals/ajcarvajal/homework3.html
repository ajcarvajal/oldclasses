Skeleton Size Dance Intensity
This skeleton will never have enough dancing
He must dance until the end of time

CONTROLS:
a,d,q,w,e   -dance

Techniques used:
Sprites: used blits to create sprite
Character animation: made a MyImageArray for each animation. Skeleton keeps track of current animation to play
Menu/Zoom: used html sliders. The zoom value is passed to the vertex shader

Modified Skeleton sprite, original by calciumtrice.tumblr.com
